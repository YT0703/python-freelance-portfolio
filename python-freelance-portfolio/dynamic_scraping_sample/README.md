# Dynamic Web Scraping Sample

## 概要
ログイン後の動的ページを想定したスクレイピングサンプルです。  
JavaScriptで構築されるページに対して、安定したデータ取得を行う方法を検証しています。

---

## 特徴
- Seleniumによるブラウザ操作
- 動的レンダリング対応（待機処理）
- テキストベースでの価格抽出
- ノイズデータ（坪単価など）の除外処理

---

## 技術スタック
- Python
- Selenium
- 正規表現（re）

---

## 工夫点
- DOM構造に依存しすぎない設計
- 実務を想定した例外処理
- 再現性を意識したロジック

---

## 注意
本コードはポートフォリオ用のサンプルです。  
特定サイトへのスクレイピングを目的としたものではありません。
